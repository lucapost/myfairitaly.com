h2. Gallery

Le immagini originali di input devono avere dimensione 420x300 oppure 300x420 in formato *.jpg.

Per elaborare le immagini seguire la seguente procedura:

* posizionarsi con la shell nella cartella sorgente del sito

  @cd ~/myfairitaly.com@

* creare la cartella images/gallery/input/nomegalleria

  @mkdir -p images/gallery/input/nomegalleria@

* copiare le immagini originali all'interno della cartella appena creata;

  @cp /path/original/images/*.jpg images/gallery/input/nomegalleria@

* posizio


Per eseguilrlo si deve entrare con la shell nella certella /images/gallery/, identificare la galleria che si vuole generare (ad esempio origin/XX_nomegalleria/ ) ed eseguire lo script,
