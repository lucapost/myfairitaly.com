h2. gestione galleria

Le immagini delle gallerie si trovato nella cartella /images/gallery/.

Le immagini originali devono avere dimensione 420x300 oppure 300x420 e formato *.jpg.

Ciascuna galleria deve avere le sue immagini originali posizionate nella cartella /images/gallery/origin/XX_nomegalleria.

All'interno della cartella /images/gallery/origin/XX_nomegalleria/, oltre alle immagini jpg, devono essere presenti i file it.txt ed en.txt.

I file it.txt ed en.txt contengono le informazione sulle immagini: descrizione e titolo.

Ogni riga di questi files corrisponde ad una immagine. Il primo campo è il nome dell'immagine, il secondo la descrizione, il terzo il titolo. Ciuscun campo deve essere separato dai due punti. Un esempio di riga è:

	01.jpg:questa è la descrizione:questo è il titolo

Il file /images/gallery/generate.sh è lo script che genera le immagini per le gallerie ed il codice html da inserire dentro [...]/gallerie.textile.

Per eseguilrlo si deve entrare con la shell nella certella /images/gallery/, identificare la galleria che si vuole generare (ad esempio origin/XX_nomegalleria/ ) ed eseguire lo script,

	./generate.sh origin/XX_nomegalleria

Una volta eseguito lo script, sul terminale comparirà il testo html da inserire nel file *.textile.
