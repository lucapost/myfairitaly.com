h2. Gallery

Le immagini originali di input devono avere dimensione 420x300 oppure 300x420 in formato *.jpg.

Per elaborare le immagini seguire la seguente procedura:

* posizionarsi con la shell nella cartella sorgente del sito:

  @cd ~/myfairitaly.com@

* creare la cartella _gallerysource/nomegalleria:
  
  @mkdir -p _gallerysource/nomegalleria@

* copiare le immagini originali all'interno della cartella appena creata:

  @cp /path/original/images/*.jpg _gallerysource/nomegalleria@

* eseguire lo script ./gallery.sh avendo cura di indicare precisamente la cartella delle immagini originali da elaborare:

  @./gallery.sh _gallerysource/nomegalleria@

* al termine dello script le immagini elaborate vengono posizionate direttamente nella cartella destinazione @./_src/images/gallery/nomegalleria@.
